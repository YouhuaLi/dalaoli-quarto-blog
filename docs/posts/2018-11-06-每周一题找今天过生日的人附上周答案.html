本期题目：找今天过生日的人

今天你过生日，但很无聊。你跟一个朋友打赌，说：“我要拦住路上每一个人，问他们的生日，直到找出一个今天也过生日的人”。你的朋友说：“算了吧，如果你在300个人以内找到这个人，算我输，我请你和那个人吃生日蛋糕”。你激动的说：“就这么决定了！”

假设路上每个人的生日是完全均匀分布的，不考虑有人2月29日生日的情况（“今天”也不是2月29日），以上打赌谁的赢面高？平均路上要找多少人可以找出一个今天过生日的？

提示：以上问题的的含义就是要有多少人后，其中有至少一个人今天过生日概率大于1/2。

有一个很著名的概率问题是：一个50人的班级，其中有两人同一天生日的概率是多少？但本题与该题不一样，不要混淆。

请直接留言回答。

上周题目：动物世界如何生存

“动物世界”…此片悬念诱人，特效绚丽，是今年国产片中的佳作，值得推荐。大老李最喜欢它的原因是它的故事背景，其设定是来自一部日本漫画，是这样一个大型“石头剪刀布”游戏：

一共有103人参加游戏，每人分得3颗星星标记，4张石头牌、4张剪刀牌、4张布牌，一共412张石头、412张剪刀、412张布。任何两人之间可以通过出牌，进行石头剪刀布游戏。输的人会输掉一颗星星给对方，用过的卡牌被销毁。打平的话则双方不会有星星的交换，卡牌仍销毁。

游戏一共进行4个小时，满足以下条件之一的会输掉游戏：
  1. 游戏结束后，星星少于3个。
  2. 游戏结束后，牌没有出完。
  3. 假如游戏过程中星星为0，就会提前出局（出局会被关小黑屋，送去做人体试验）。

现在假设两人各持12张牌，以完全随机出牌的方式，互相决斗，直到某人出局(星星数降为0)或两人卡牌用完，则：

游戏平均会进行多少轮？两人都暂时幸存的概率是多大（即有两人卡牌用完，但两人至少有1颗星星）？

附加：如果两人一开始的卡牌数分别是p和q，则以上问题答案会如何？

上周题目意外的难。，主要难度在于两人出牌的进度过程会影响后续的胜率。目前我只收到署名杨元双的听众，进行的十分详尽的计算。其计算的结果为：平均进行6.48轮，两人都获胜的概率为89%。但似乎与我的程序模拟结果不符。

（杨元双听众给我发来的部分解答，非常仔细的计算 ）

大老李编了一个python程序进行验证，程序代码可在这里下载：http://t.cn/Ew1mGK0

程序模拟了10万场两人对战，运行了三次结果如下：

python aw.py {3: 7731, 4: 7514, 5: 7684, 6: 7015, 7: 6378, 8: 5799, 9: 5282, 10: 4825, 11: 4452, 12: 43320} 平均游戏轮数：8.893970 两人都幸存的概率：0.314710 python aw.py {3: 7800, 4: 7645, 5: 7481, 6: 6888, 7: 6359, 8: 5760, 9: 5443, 10: 4876, 11: 4345, 12: 43403} 平均游戏轮数：8.896840 两人都幸存的概率：0.314130 python aw.py {3: 7692, 4: 7621, 5: 7609, 6: 7045, 7: 6401, 8: 5826, 9: 5317, 10: 4688, 11: 4365, 12: 43436} 平均游戏轮数：8.892700 两人都幸存的概率：0.315540

以上可以看出游戏平均轮约8.89轮，但具体轮次来看，打满12轮的概率最高。两人都幸存的概率约31%。
