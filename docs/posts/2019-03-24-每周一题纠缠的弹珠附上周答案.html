上周题目：平均数序列

有以下5个数：

45, a,b,c,d,100

从b开始，每个数都是前两个数的平均数，问a是多少？

答案是：125。此题只要耐心点解方程即可，借用读者ouou的留言：

根据条件可得：

45 + a = 2b – （1）

a + b = 2c – （2）

b + c = 2d – （3）

c + d = 200 – （4）

将4式左右相加，得

45 + 2a + 2b + 2c + d = 2b + 2c + 2d + 200

45 + 2a = d + 200

2a - d = 155 – （5）

根据（1）式，得

a = 2b - 45

根据（4）式，得

d = 200 - c

将 a = 2b - 45 和 d = 200 - c 带入（5）式，得

2 *（2b - 45）-（200 - c） = 155

4b + c = 445 – （6）

将 d = 200 - c 带入（3）式，得

b + c = 2 * ( 200 - c )

b + 3c = 400 – （7）

联立（6）式和（7）式

解得 b = 85

根据（1）式，得 a =125

上期加餐题：自定义的骰子

你和朋友一起玩掷骰子比大小的游戏，如果打平则重掷。你有一个便利条件是你可以自由安排一个骰子点数的位置：

即你可以将21个黑点任意分配到6面上，不能有负数或分数的点数。

请问：是否存在一种点数分配方案，使得你的骰子比对手的标准骰子有优势？如果有，胜率最高可以达到多少？点数是如何安排的？

答案：是存在更好的方案的。

有些读者认为是“不能”，理由是不管点数如何分配，每次掷骰值的期望值总是21/6=3.5，所以不会更好。但是掷骰比赛并不是比“期望值”的比赛，如果你的骰子是（0,0,0,0,0,21）的点数分配，肯定是不利的，虽然你的点数期望值仍然是3.5。

因此，我们可以考虑一种极端情况，比如，将点数为“1”的那个点，移到“5”那里，得到(0,2,3,4,6,6)的分配。这样，我们失去了1/36的打平的机会，但是直觉上，你会感觉到你获胜和打平的机会增加了不止1/36。具体胜率表如下：

所以，(0,2,3,4,6,6)骰子的胜率是16/31。为找到最佳点数分配方案，接下来，我们从两个标准骰子的对阵表开始分析：

蓝色是我方的骰子，并且定义某一行的“分值”为：

某行的分值=本行我方获胜的格子数-本行对方获胜的格子数

考虑这样一个问题，是否存在一种方法，当我移动一个黑点到另一面时，可以使得某一行的分值增加数大于2？稍加思考，你会发现这是不可能的。比如当4变5的时候，本方增加了一格获胜，对方减少了一格获胜，分值增加等于2。如果被增加的面本来就是6，则分值只增加1。总之，不可能存在分值增加大于2的情况。

同理，也不存在移动1点后，某行分值损失多于2的情况。

此时我们会发现移动点数较少的面上的点到其他面，并且使其他面上的点不超过6，总是划算的。也就是要使点数为0的尽可能多，但是使其他面不超过6。因此得到如下三种方案：

0,0,3,6,6,6

0,0,4,5,6,6

0,0,5,5,5,6

其胜率都是17/32！

有人指出，这个游戏很像“田忌赛马”，的确如此。

另外，还可以考虑，你知道对方的投掷是如此自定义的，你有没有可能设计出一种骰子，可以胜过对手？是否存在一种方案，使得对方无论如何改进其方案，都无法获得更大收益？根据纳什均衡理论，这种方案肯定是存在的。而在本游戏中，因为双方地位相等，因此这种方案的效果必然是：无论对方如何设计，我方骰子的胜率大于等于1/2。具体方案供读者自行思考。
